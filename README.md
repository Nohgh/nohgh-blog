# Nohgh Blog

안녕하세요. 노기훈 블로그입니다.

### script

```ts
// 개발 서버 실행
pnpm dev

// RSS 생성 후 프로덕션 빌드
pnpm build

// 프로덕션 서버 실행
pnpm start

// 테스트 실행
pnpm test

// 테스트 커버리지 확인
pnpm test:coverage

// 코드 린트 검사
pnpm lint

// 코드 린트 자동 수정
pnpm lint:fix

// 코드 포맷팅
pnpm format

// 코드 포맷 검사
pnpm format:check

// TypeScript 타입 검사
pnpm type:check

// QR 코드 생성
pnpm qr

// 블로그 CLI 도구
pnpm blog

// RSS 피드 생성
pnpm generate:rss
```
